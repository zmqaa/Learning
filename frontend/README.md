# LearnLab Frontend

AI 驱动的备考学习工具前端。React 19 + TypeScript + Vite 8 构建，支持亮/暗主题切换。

## 开发

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # 生产构建 → dist/
npm run preview   # 预览生产构建
```

## 技术栈

- React 19 + TypeScript
- Vite 8
- react-router-dom v7（SPA 路由）
- react-markdown + KaTeX（Markdown 渲染 + 数学公式）
- CSS 变量主题切换（暖米纸亮色 / 墨绿终端暗色）
